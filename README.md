Family Price Chart Generator
This project visualizes the national price ordering of input families (e.g., construction materials) across Brazilian states.

It creates a clean report-style visualization, with regions grouped under state labels and an integrated color-coded table as the main legend.

Built with: Python, Google Colab, Matplotlib, Seaborn, Pillow, Pandas + R & Rstudio

Outputs: .png file containing chart + summary table

Example: Ordering_Family_Washed Sand.png
